# Contact List App

## Features
- **Add Contacts**: Users can create new contacts by entering their name, phone number, company, and email address.
- **View Contact List**: All added contacts are displayed in a list format, making it easy to view each contact's information.
- **Search Contacts**: Users can search for contacts by name using the built-in search functionality.
- **Data Persistence**: All contacts are saved using `UserDefaults`, ensuring that data is retained even after the app is closed.
